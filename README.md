# Front Dummy

> Front-End project kick-starter

## Dev stack:

- PostCSS
- React
- Gulp
- Webpack
- ESLint


## Documentation

- CSSG
- SC5


## Install

```
npm i
```

### Optional

```
npm dedupe
```


## Build modes

```
gulp
```

### Production

```
gulp build [--p]
```

### Publish

```
gulp build [--p]
gulp publish
```


## Styleguides with SC5

```
gulp styleguide
```

### Dev mode

```
gulp styleguide-dev
```
